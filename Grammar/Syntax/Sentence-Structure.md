# Sentences structure 
---
All sentences follow the structure of **SVO**, but a more accurate may looks like: 

```
Linker, Context position, Subject, Verb, Adverb, Direct Object, Indirect Object
```
---
## Linker
Linker are prepositions (visit the prepositions page for more information). 

---
## Context position
Nouns, adverbs and adjectives to locate where the action is occurring, this can refer in time or position contexts: 
- "**Daifore**, Mi ni kone fu li kasaca" -> "**Tomorrow**, I will return home"

 Also you can establish frequency in where the action is done: 
 - "**Nani daite**, mi ni kore ne daitese" -> "**Some days**, I run in the morning"

---
## Subject 
Subjects are nouns and must be capitalized 
- "**Tu** no ni kore" -> "**You** don't run"
- "daitese, **Kanao** mio ni kome pa na" -> "My **dog** was eating this morning"

Subjects can have adjectives and prepositions, but always have to be first
- "nani cika, **kibodo** regosa biga mia ni nese ke fosu" -> "My red big **keyboard** need a fix some times"
